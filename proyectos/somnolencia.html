Detección de Somnolencia con Transformers RT-DETR

Visión por Computadora en Tiempo Real para Conductores

Proyecto basado en una arquitectura Transformer optimizada para detectar somnolencia mediante análisis ocular en tiempo real. Entrenado sobre infraestructura HPC con GPU NVIDIA A100 y desplegado en una Raspberry Pi 3 B+.

Resumen del Proyecto

Este sistema de visión artificial detecta estados de somnolencia mediante un modelo Transformer RT-DETR, capaz de identificar ojos abiertos/cerrados con gran precisión incluso en condiciones difíciles (nocturnidad, gafas, accesorios faciales).

  • ⭐ Modelo RT-DETR optimizado para clasificación de ojos.
  • ⭐ Entrenado sobre GPU NVIDIA A100 (40 GB) en la infraestructura HPC CEDIA.
  • ⭐ Dataset propio 128×128 px balanceado vs desbalanceado.
  • ⭐ Deploy en Raspberry Pi 3 B+ con inferencia en tiempo real.
  • ⭐ 95.21% de precisión en ambientes diurnos y nocturnos.

Tecnologías Utilizadas

Python PyTorch Ultralytics RT-DETR Transformers OpenCV NVIDIA A100 HPC CEDIA Raspberry Pi

Resultados del Modelo

Métricas Principales

  • Exactitud general: 95.21%
  • Precisión: 95.47%
  • Recall: 95.75%
  • F1 Score: 95.61%

Rendimiento en condiciones reales

  • Gafas normales: 94.58% precisión
  • Gafas de sol: 94.32% precisión (muy destacado vs otros modelos)
  • Día: 91.35% F1-score
  • Noche: 92.53% F1-score
