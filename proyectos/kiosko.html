KioskoH2M – API REST para gestión de negocios

Sistema backend rápido, modular y escalable construido con FastAPI

API diseñada para la administración de ventas, clientes, productos, asistencias y pagos en pequeños negocios como restaurantes, comedores y kioscos. Forma parte del ecosistema DeepVisionH2M.

Descripción del Proyecto

KioskoH2M API es una solución backend moderna que centraliza la gestión de operaciones comerciales para pequeños negocios. Su modelo de datos contempla la administración de personas, empresas, productos, ventas, pagos, categorías y asistencia, manteniendo integridad y estados históricos.

Está construida con FastAPI y SQLModel, garantizando validación sólida, consultas rápidas y escalabilidad. Pensada para integrarse con aplicaciones web, móviles o sistemas POS.

Características Principales

Gestión de personas
Gestión de empresas
Productos y categorías
Ventas y facturación
Deudas y abonos
Control de asistencia
Modelo relacional robusto
Alto rendimiento con FastAPI
Estados activos/inactivos

Tecnologías Utilizadas

FastAPI Python SQLModel SQLite Pydantic Uvicorn REST API

Arquitectura del Sistema

Modelo modular basado en capas: Routes → Services → Repository → Modelos SQLModel → Base de datos.

Ejemplo de Endpoints

# Obtener todos los productos
GET /productos/

# Registrar una venta
POST /ventas/

# Registrar un pago o abono
POST /pagos/

# Obtener clientes con saldo pendiente
GET /clientes/deudas/
